# --- Hugo Blog Workflow (Updated) ---

1. Create a new post:
   hugo new content/posts/your-post-name.md

2. (Optional) Preview locally:
   hugo server -D

3. Deploy (via GitHub Actions):
   git add .
   git commit -m "Your commit message"
   git push

Note: 
- The local 'hugo' build is no longer necessary for deployment.
- GitHub Actions in '.github/workflows/hugo.yaml' handles the production build and deploy.
- Remember to update 'baseURL' in 'hugo.toml' to your actual blog URL.
